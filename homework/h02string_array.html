문자열과 배열 실습

  1. "안녕"=="안녕 : 동등비교연산은 값을 비교하는 연산이기 때문에 기본데이터로 true를 반환
  2. "안녕"==new String("안녕") : 자료형에서 값이 존재하면(랩퍼클래스) 값을 빼와서 비교한다. true
  3. "안녕"===new String("안녕") : 완전동등비교연산 기본형과 자료형을 비교했기 때문에 false를 반환
  4. ['안','녕']==['안','녕'] : 자료형은 값이 존재하지 않기 때문에 비교연산을 할 수 없다.
  5. new String("안녕")==new String("안녕") :랩퍼클래스의 자료형은 값을 참조하기 때문에 값을 비교해서 true여야 하지만 결과는 false, 동등비교연산에 값이 존재해야 자료형에서 값을 찾는다.
  6. 사과,배,자두,딸기 를 순서대로 저장하는 배열 자료를 생성해 보세요. let arr=["사과",'배',"자두","딸기"]
  7. 위 자료에서 index가 2인 자료를 출력하세요. document.write(arr[2])
  8. 위 자료에 '수박'을 추가하세요. arr.push("수박"); arr[4]="수박"
  9. 위 자료의 길이를 출력하세요. arr.length
  10. typeof "안녕" : 결과가 무엇인가? "string" typeof 연산자는 기본형은 각자 이름(boolean,number)으로 반환한다.
  11. typeof new String("안녕") : 모든 자료형을 object로 반환한다.
