함수를 정의하고 호출해 보자

  • function nameName(매개변수){지역scope} : 함수 정의법
  • 함수를 정의하면 코드와 상관없이 자바스크립트 엔진이 함수를 먼저 메모리에 저장하고 코드를 실행한다. (코드는 항상 위에서 아래로 순서대로 실행된다. 때문에 아래서 선언된 것은 위에서 호출 불가능)
  • 때문에 아래에 정의한 함수를 위에서도 호출할 수 있는 함수 호이스팅 발생한다.
  • (a, b) : 매개변수는 let을 생략하고 지역변수가 된다.
  • {let c=10} :함수의 지역변수는 let을 생략하면 안된다.(매개변수와 다른이름 작성)
  • {return} : return 을 하면 함수가 종료된다.
  • {return data} : data를 반환하고 함수가 종료된다.
  • {} : return 이 없는 함수를 void 함수라 한다.
