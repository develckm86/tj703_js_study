존재하는 데이터를 판별하는 ||과 ??

"",0,false,null,undefined 데이터를 제거하는 || 연산

let 대입 = a || b|| c || d || c || ...

|| 연산은 모든 연산중에 하나만 true만 되어도 모두가 true기 때문에 true가 나올때 까지 연산하는데 이를 이용해 존재하는 데이터가 true 라서 존재하지 않는 데이터 중에 존재하는 데이터를 찾을 때 사용한다.

단점 : "", 0 를 찾을 수 없다.(false기 때문)

null,undefined 데이터를 제거하는 ?? 널병합 연산

|| 연산이 데이터의 값을 비교해서 "",0,false 제거하기 때문에 ?? 연산을 새롭게 만들어서 제공
