type과 class 그리과 객체 실습

type을 정의해서 객체를 생성하고 재사용하는 법을 객체지향 문법이라 한다.

  • 문제1: new String("안녕"),new Number(55) : 이때 "안녕"과 55는 무엇의 무엇값이라 부를 수 있나요?
  • 정답 : "안녕"과 55는 객체의 초기값(생성자가 정의하는 개체의 초기값이다.)
  • 초기값 : 객체의 필드인데 최초에 정의된 것
  • 문제2 : 자동차(Car) 클래스를 만들고 싶은데 필드로 이름(name)과 모델번호(modelNum)를 초기값으로 갖게 정의하세요.
  • class Car{ constructor(){ this.name="경민의 제네시스"; this.modelNum="gv80" } }
  • 문제3 : 자동차 클래스로 객체를 생성하고 콘솔에 출력해 보세요.
  • 정답:
    let car=new Car();
    console.log(car)
  • 자동차 클래스의 생성자에 매개변수로 이름을 받고 필드로 정의한 이름을 매개변수 값으로 바꾸세요. :생성자로 이름의 초기값을 정의하세요.(만들때 이름이 다른차면 좋겠다.)
  • 저동차 클래스를 만들때 이름을 변경해서 생성해 보고 콘솔에 출력도 해보세요!
